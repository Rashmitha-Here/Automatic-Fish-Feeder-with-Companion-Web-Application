# Automatic-Fish-Feeder-with-Companion-Web-Application
The Automatic Fish Feeder with Companion Web Application is an IoT-based solution designed to automate and manage fish feeding schedules remotely. The system integrates a web-based dashboard with an Arduino-controlled feeding mechanism, enabling scheduled and controlled feeding without manual intervention.
